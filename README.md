# AI Stock News Sentiment Analyzer

This project analyzes financial news and determines whether the sentiment is positive or negative using Natural Language Processing (NLP).

It helps traders understand market sentiment based on news headlines.

## Features

* AI-powered sentiment analysis
* Financial news classification
* NLP using transformer models
* Simple and fast predictions

## Technologies Used

* Python
* Transformers
* PyTorch

## Installation

Install required libraries:

pip install transformers torch

## How to Run

1. Clone the repository

git clone https://github.com/yourusername/stock-news-ai

2. Go to the project folder

cd stock-news-ai

3. Run the script

python stock_sentiment_ai.py

## Example

Input:

"Bitcoin price is expected to rise this year"

Output:

Positive sentiment
